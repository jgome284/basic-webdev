Inline-flex

display: inline-flex
The display: inline-flex property creates an inline-level flex container. It allows the container to be displayed inline, while still maintaining flexbox behavior for its children.

Flex

display: flex
The display: flex property creates a block-level flex container. It enables a flex context for all its direct children, which are called flex items. Flex items can be aligned, distributed, and ordered along the main and cross axes of the container. flex containers have two axes: a main axis and a cross axis. By default, the main axis is horizontal and the cross axis is vertical.

The main axis is used to position flex items with the following properties:
  • justify-content
  • flex-wrap
  • flex-grow
  • flex-shrink

The cross axis is used to position flex items with the following properties:
  • align-items
  • align-content

Flex Start (Horizontal)

justify-content: flex-start
The justify-content: flex-start property aligns the flex items at the beginning of the container along the main axis. This means that the items are packed flush to each other toward the edge of the alignment container depending on the flex containerâ€™s main-start side. This is the default value for the justify-content property.

Flex End (Horizontal)

justify-content: flex-end
The justify-content: flex-end property aligns the flex items at the end of the container along the main axis. This means that the items are packed flush to each other toward the edge of the alignment container depending on the flex containerâ€™s main-end side.

Center (Horizontal)

justify-content: center
This aligns the flex items at the center of the container along the main axis. This means that the items are packed flush to each other toward the middle of the container.

Space Around (Horizontal)

justify-content: space-around
This distributes the items evenly within the container along the main axis. The spacing between each pair of adjacent items is the same. The empty space before the first and after the last item equals half of the space between each pair of adjacent items.

Space Between (Horizontal)

justify-content: space-between
This distributes the items evenly within the container along the main axis. The first item is flush with the start edge, and the last item is flush with the end edge. The spacing between each pair of adjacent items is the same.

Flex Start (Vertical)

align-items: flex-start
This aligns the flex items vertically up, flush against the flex containerâ€™s main-start or cross-start side.

Flex End (Vertical)

align-items: flex-end
This aligns the flex items vertically down, flush against the flex containerâ€™s main-end or cross-end side.

Center (Vertical)

align-items: center
This aligns the flex items at the center of the container along the cross (vertical) axis. The items are packed flush to each other toward the middle of the container.

Baseline (Vertical)

align-items: baseline
This aligns the flex items such that their flex container baselines align. The item with the largest distance between its cross-start margin edge and its baseline is flushed with the cross-start edge of the line.

Stretch (Vertical)

align-items: stretch
This stretches the flex items to fill the container, respecting the itemsâ€™ width and height limits. If the items are smaller than the alignment container, auto-sized items will be equally enlarged to fill the container.

Flex-Grow

flex-grow: <factor>
The flex-grow: <factor> property specifies how much a flex item will grow relative to the rest of the flex items in the same flex container. The <factor> is a positive number that represents the proportion of the available space that the item should take up. For example, if all items have flex-grow: 1, they will share the space equally. If one item has flex-grow: 2, it will take up twice as much space as the others. This property only applies to flex items, not to the flex container itself.

Flex-Shrink

flex-shrink: <factor>
The flex-shrink: <factor> property specifies how much a flex item will shrink relative to the rest of the flex items in the same flex container. The <factor> is a positive number that represents the proportion of the space that the item should give up when the container is too small. For example, if all items have flex-shrink: 1, they will shrink equally. If one item has flex-shrink: 2, it will shrink twice as much as the others. This property only applies to flex items, not to the flex container itself3.

Flex-Basis

flex-basis: <length>
The flex-basis: <length> property specifies the initial size of a flex item along the main axis, before any remaining space is distributed according to the flex-grow and flex-shrink properties. The <length> can be a fixed value, a percentage, or one of the keywords auto, content, max-content, min-content, or fit-content. This property only applies to flex items, not to the flex container itself

Flex (Shorthand)

flex: <flex-grow> <flex-shrink> <flex-basis>
The shorthand flex property provides a convenient way for specifying how elements stretch and shrink, while simplifying the CSS required. The flex property allows you to declare flex-grow, flex-shrink, and flex-basis all in one line.

Align Content

Flex Start

align-content: flex-start
All rows of elements will be positioned at the top of the parent container with no extra space between.

Flex End

align-content: flex-end
All rows of elements will be positioned at the bottom of the parent container with no extra space between.

Center

align-content: center
All rows of elements will be positioned at the center of the parent element with no extra space between.

Space Between

align-content: space-between
All rows of elements will be spaced evenly from the top to the bottom of the container with no space above the first or below the last.

Space Around

align-content: space-around
All rows of elements will be spaced evenly from the top to the bottom of the container with the same amount of space at the top and bottom and between each element.

Stretch

A conditional state...
If a minimum height or no height is specified, the rows of elements will stretch to fill the parent container from top to bottom (default value)

Flex Direction

The main axis and cross axis are interchangeable. We can switch them using the flex-direction property. If we add the flex-direction property and give it a value of column, the flex items will be ordered vertically, not horizontally.

Row

flex-direction: row
Elements will be positioned from left to right across the parent element starting from the top left corner (default).

Flex Flow

flex-flow: <flex-direction> <flex-wrap>
Like the shorthand flex property, the shorthand flex-flow property is used to declare both the flex-wrap and flex-direction properties in one line. The first value in the flex-flow declaration is a flex-direction value and the second is a flex-wrap value. All values for flex-direction and flex-wrap are accepted.
